# Disku

`disku` is a cli tool for disk usage statistics. identify large files and directories.

## Install

You can install this crate with Cargo by running:

```bash
> cargo install disku
```

## Demo

![Demo](https://github.com/matinturkaman/disku/blob/main/screenshot.png)

## How to use

You can specify the target path for disk usage analysis.

```bash
> disku ~/
```
